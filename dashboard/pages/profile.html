💾 Backup & restore — encrypted .ghs-bundle export / import
Pack the entire profile (DB rows, user-data-dir, vault items, cookie pool entries, fingerprint history, canary timeline) into a single encrypted bundle. Restore on a different machine to keep the same browser identity — same fingerprint, same cookies, same logins. The master password is required to encrypt and is never stored — lose the password, lose the bundle.
Cloud sync: —
⬇ Download encrypted backup
The bundle is encrypted with a master password using PBKDF2-HMAC-SHA256 (200k iterations) + Fernet (AES-128-CBC + HMAC-SHA256). Use the SAME password as your vault master — that way vault items in the bundle stay decryptable on the destination host. The password is never stored or transmitted in plaintext to disk.
⬆ Restore from .ghs-bundle
Select an encrypted bundle file produced by Ghost Shell's Download backup action. The file extension should be .ghs-bundle; bundles produced by an older Ghost Shell may use .gsbk. The header preview below is unencrypted — no password is sent until you click Restore.

⚠ Danger zone — destructive operations scoped to this profile
These operations affect only the currently-selected profile. They do not touch the proxy library, scripts, or global config. Delete cannot be undone — the user-data-dir is wiped from disk.
Reset consecutive-blocks counter
Clears the "N runs in a row hit a captcha/block" counter. Use after you've rotated the proxy or edited the fingerprint and want a clean slate — the health-based pause-on-block logic will stop flagging this profile. No historical data is lost.
Clear session-quality history
Wipes the per-run quality log (ad counts, captcha hits, durations) for this profile. Run records themselves stay in the Runs page — only the session-quality rollup is cleared. Cannot be undone.
Delete profile
Removes the profile from the database and deletes its entire user-data-dir from disk — cookies, session, history, fingerprints, all gone. Proxy assignments to this profile are released. Cannot be undone. Any running job for this profile is stopped first.
