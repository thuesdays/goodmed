⚠ Danger zone — destructive operations scoped to this profile
These operations affect only the currently-selected profile. They do not touch the proxy library, scripts, or global config. Delete cannot be undone — the user-data-dir is wiped from disk.
Reset consecutive-blocks counter
Clears the "N runs in a row hit a captcha/block" counter. Use after you've rotated the proxy or edited the fingerprint and want a clean slate — the health-based pause-on-block logic will stop flagging this profile. No historical data is lost.
Clear session-quality history
Wipes the per-run quality log (ad counts, captcha hits, durations) for this profile. Run records themselves stay in the Runs page — only the session-quality rollup is cleared. Cannot be undone.
Delete profile
Removes the profile from the database and deletes its entire user-data-dir from disk — cookies, session, history, fingerprints, all gone. Proxy assignments to this profile are released. Cannot be undone. Any running job for this profile is stopped first.
