Post-click actions
What Ghost Shell does after finding an ad — simulate a real user
🎯 Default pipeline 0
Runs for every competitor ad detected. Steps execute in order. Drag rows to reorder. Each step has a "probability" (0-1) — e.g. 0.3 means "only 30% of the time this ad gets a click".
⭐ On target-domain ads 0
Runs instead of the default pipeline when the ad's domain matches one in Search → Target domains. Typically lighter — you don't want to click your own ads excessively (that costs you real money via CPC).
