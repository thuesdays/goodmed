Encrypted storage for logins, crypto wallets, API keys, TOTP secrets — everything that needs to stay off disk in plaintext.

Set a master password
This password encrypts everything you store here. It is never written to disk — if you forget it, your secrets are gone. There is no recovery. Pick something memorable but strong.
⚠
We derive a Fernet key from this password with PBKDF2-HMAC-SHA256 (200k iterations). Stored secrets can only be opened from a session that knows the password. Nothing fancier than that — this is local-only, not a cloud service.
