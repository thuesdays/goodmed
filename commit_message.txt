Initial public release: Ghost Shell Anty v0.2.0

A self-hosted antidetect Chromium browser with a built-in Flask
dashboard, designed for Google Ads competitive intelligence and
serious automation work. Local-first, source-open, $0/month --
built as an alternative to Multilogin / GoLogin / Dolphin{anty}.

What ships in this initial release
==================================

* Patched Chromium 149.0.7805 with C++-level stealth modifications
  (no JS injection that detection scripts can fingerprint).
* Per-profile fingerprint generator with a 35/35 coherence
  selfcheck (timezone <-> locale <-> Accept-Language, GPU <-> canvas,
  fonts <-> OS, etc).
* Modern device template catalog: RTX 4070/4080/4090, 64GB+ RAM
  presets, MacBook Pro M3, Threadripper.
* Mobile profiles with CDP-driven device emulation (touch,
  swipe-scroll, DeviceMotion / DeviceOrientation APIs).
* Dual-mode profiles -- flip a single profile between desktop and
  mobile without re-creating it.
* Per-profile proxy + identity overrides; bulk actions on the
  Profiles page (start/stop/delete N at a time).
* Cookie pool + warmup engine with site presets (Google search,
  YouTube) -- gradual heat-up before the real workload.
* Cron-style scheduler with live preview of the next 5 fire times
  and process-isolated runs.
* Vault: Fernet-encrypted credentials store, key derived via
  PBKDF2-HMAC-SHA256 (200k iters); TOTP (2FA) generator with
  30-second auto-refresh.
* Competitor monitor: per-query result tracking with snapshot
  diffing; visual diff of competitor ad copy + URL changes over
  time.

Distribution
============

End users: download GhostShellAntySetup.exe from the Releases page.
Source-only checkouts: run scripts/download_chromium.ps1 to fetch
the matching pre-built Chromium asset.

The installer pipeline lives in a separate private repo:
ghost_shell_browser_inno.

Tech stack
==========

Python 3.11+ (3.13 recommended), Flask, SQLite (18 tables),
patched Chromium 149.0.7805. License: MIT.
